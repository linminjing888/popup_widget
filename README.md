# popup_widget

A pop-up bubble text component, lightweight, easy to use.

## Getting Started

![img1](popup_one.png) 

![img2](popup_two.png) 

```
PopupWidget(
    child: Icon(Icons.help_outline, size: 16, color: Colors.blue),
    text: "Lord, I'm one, Lord",
    contentSize: Offset(16, 16),
);
```
